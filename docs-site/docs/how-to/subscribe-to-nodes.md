# How to Subscribe to Nodes

Subscribing to a node adds it to your monitored list. Subscribed nodes appear on the map and can be filtered with the "Only show subscribed nodes" option.

## Subscribe from the Node List

1. In the **Node List** panel on the right, find the node you want to monitor.
2. Check the **checkbox** on the left side of the node card.
3. The node is now subscribed. It will appear on the map.

## Unsubscribe from a Node

1. Find the subscribed node in the Node List.
2. Uncheck the **checkbox** on the node card.
3. The node is removed from your subscriptions and will no longer appear on the map.

## What Subscriptions Do

- **Map**: Only subscribed nodes are shown on the map.
- **Filtering**: Use the "Only show subscribed nodes" filter to limit the list to your subscriptions.
- **Updates**: Subscriptions are saved to your account and persist across sessions.
