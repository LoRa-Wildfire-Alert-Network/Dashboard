# How to Use the Map

The map shows the locations of your subscribed nodes. Each marker is color-coded by status.

## Map Marker Colors

| Color | Meaning |
|-------|---------|
| **Red** | Smoke detected — potential fire alert |
| **Orange** | Low humidity (below 15%) or high temperature (above 35°C) |
| **Yellow** | Low battery (below 20%) |
| **Green** | Normal — no smoke, battery OK, humidity OK, temperature OK |

## Interacting with the Map

- **Pan**: Click and drag to move the map
- **Zoom**: Use the scroll wheel or the +/- controls
- **Click a marker**: Expands that node's details and centers the map on it. The corresponding node card in the list also expands
- **Click an expanded marker again**: Collapses it

## Map and Node List Sync

When you click a marker:

- The map zooms to that node
- The matching node card in the list expands to show full details (temperature, humidity, battery, smoke status)
- The Node Details panel on the left shows current and historical data for that node

When you click a node card:

- The map zooms to that node's marker (if it's on the map)
- The marker appears larger to show it's selected
- The Node Details panel updates for that node
