# How to Filter Nodes

Use filters to focus on nodes that match specific conditions.

## Opening the Filter Panel

1. Click the **filter icon** (funnel) next to the Node List title.
2. The filter options appear below.

## Filter Options

### Only show subscribed nodes

When checked, the Node List shows only nodes you've subscribed to. The map always shows only subscribed nodes.

### Alerts

- **Smoke Detected** — Show only nodes where smoke has been detected.
- **Temp above** — Show nodes with temperature above a value you enter (in °C).
- **Humidity below** — Show nodes with humidity below a value you enter (in %).

### Node Health

- **Low battery (&lt; 20%)** — Show only nodes with battery below 20%.

## Using the Filters

1. Check or uncheck the options you want.
2. For **Temp above** or **Humidity below**, check the option first, then enter a number in the field that appears.
3. The Node List updates immediately to show only matching nodes.
4. Click the filter icon again to hide the filter panel.

## Combining Filters

All active filters work together. For example, you can show only subscribed nodes that have smoke detected and low battery.
