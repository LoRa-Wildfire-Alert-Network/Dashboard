# LoRa Wildfire Dashboard

Welcome to the LoRa Wildfire Dashboard. Use the dashboard to monitor LoRa-based wildfire sensor nodes in real time. You can view sensor data on the map, subscribe to nodes you care about, and filter by various conditions.

## What You Can Do

- **Sign in** with your account to access the dashboard
- **Subscribe to nodes** to track them on the map
- **View the map** to see node locations with color-coded status
- **View node details** in the left panel — current readings (RSSI, SNR, altitude, gateway) plus the last 50 historical readings
- **Filter nodes** by smoke detection, temperature, humidity, battery level, or subscription status
- **Expand node cards** to see detailed sensor readings (temperature, humidity, battery, smoke status)
- **Create and manage organizations** — invite team members, assign roles, and switch between organizations
- **Manage your account** — update profile picture and username, add 2FA, create passkeys

## Quick Start

1. [Sign in or create an account](getting-started/sign-in.md) — if you signed up with username only, [add authentication](getting-started/complete-account-setup.md)
2. [Subscribe to nodes](how-to/subscribe-to-nodes.md) you want to monitor
3. Click a node to open the [Node Details](how-to/view-node-details.md) panel
4. Use the [map](how-to/use-the-map.md) and [filters](how-to/filter-nodes.md) to focus on what matters
5. [Create an organization](how-to/organizations.md) and invite your team
