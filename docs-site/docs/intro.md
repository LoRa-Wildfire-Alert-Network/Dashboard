# LoRa Wildfire Dashboard

Welcome to the LoRa Wildfire Dashboard. Use the dashboard to monitor LoRa-based wildfire sensor nodes in real time. You can view sensor data on the map, subscribe to nodes you care about, and get notified when conditions are met.

## What You Can Do

- **Sign in** with your account to access the dashboard
- **View alerts** — all signed-in users can see system alerts (smoke detected, high temp, low battery, fire risk)
- **Acknowledge alerts** — subscribers can mark alerts as reviewed
- **Subscribe to nodes** to track them on the map and receive email alert notifications
- **View the map** to see subscribed node locations with color-coded status
- **View node details** — current readings (temp, humidity, battery, smoke, RSSI, SNR, altitude, gateway) plus the last 50 historical readings and node-specific alerts
- **Filter nodes** by smoke detection, temperature, humidity, battery level, or subscription status
- **Expand node cards** to see detailed sensor readings and subscribe/unsubscribe
- **Create and manage organizations** — invite team members, assign roles, and configure member permissions
- **Manage your account** — update profile picture and username, add 2FA, create passkeys

## Quick Start

1. [Sign in or create an account](getting-started/sign-in.md) — if you signed up with username only, [add authentication](getting-started/complete-account-setup.md)
2. [Subscribe to nodes](how-to/subscribe-to-nodes.md) you want to monitor (this also enables [alert notifications](how-to/alerts.md))
3. Click a node to open the [Node Details](how-to/view-node-details.md) panel
4. Use the [map](how-to/use-the-map.md) and [filters](how-to/filter-nodes.md) to focus on what matters
5. [Create an organization](how-to/organizations.md) and invite your team
